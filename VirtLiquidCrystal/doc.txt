Here are the public methods defined in the LCD class:

LCD() - Constructor

begin() - Initializes the LCD

clear() - Clears the display

home() - Resets cursor to start

noDisplay() - Turns display off

display() - Turns display on

noBlink() - Turns cursor blink off

blink() - Turns cursor blink on

noCursor() - Hides cursor

cursor() - Shows cursor

// print() - Writes text to LCD

write() - Writes data to LCD

command() - Sends command to LCD

setBacklight() - Sets backlight level

setBacklightPin() - Configures backlight pin

autoscroll() - Turns autoscroll on/off

leftToRight() - Sets left-to-right entry mode

rightToLeft() - Sets right-to-left entry mode

scrollDisplayLeft() - Scrolls display left

scrollDisplayRight() - Scrolls display right

createChar() - Creates custom character

setCursor() - Sets cursor position
